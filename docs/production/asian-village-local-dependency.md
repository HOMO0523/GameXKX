# Asian Village local dependency

GameXXK uses the licensed Fab product [Stylized Eastern Village](https://www.fab.com/listings/9841fee2-683f-4e68-adb8-bafec270a251). The source is the user's owned UE 5.4 local installation at `D:\UE5 demo\zzz\我的项目\Content\Asian_Village`.

The vendor files are a local-only dependency and are Git ignored. Collaborators must acquire the product from Fab under their own authorized access and install it at the exact project-relative path `Content/Asian_Village`.

Do not upload the vendor assets to Tripo, GPT image-generation services, or any other external AI service. GameXXK-owned scripts, reports, maps, and adaptation assets remain normally tracked outside the ignored vendor directory.

This repository policy contains no original Fab binaries, images, or extended store copy.
